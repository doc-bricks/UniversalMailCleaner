What this companion does and does not do.

  • Reads local JSON only. No IMAP, Gmail API, OAuth, or Drive access.

  • Offline cache keeps only the last loaded secrets-free profile export.
  • Mail cleanup remains a desktop-only action in UniversalMailCleaner.
